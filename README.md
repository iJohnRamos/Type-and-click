# Type and Click

Um projetinho simples que fiz enquanto aprendia JavaScript.

A ideia é bem direta: você digita algo, clica no botão e o texto aparece na tela.
Dá pra escolher quantas vezes ele repete (1x, 2x, 4x), tem contador de palavras e também um botão pra limpar tudo.

## O que tem no projeto

* Adicionar texto digitado
* Repetir o texto com botões (1x, 2x, 4x)
* Contador de quantas palavras foram criadas
* Botão de limpar tudo
* Botões que ficam ativos quando selecionados

## Tecnologias usadas

* HTML
* CSS
* JavaScript

## Observação

Fiz esse projeto pra praticar lógica, manipulação do DOM e eventos em JavaScript.
Ainda é simples, mas já me ajudou a entender bastante coisa na prática.
